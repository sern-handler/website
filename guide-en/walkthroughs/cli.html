# sern CLI

Setting up the CLI is easy.

npm install -g @sern/cli
1

To start a new project, run:

sern init
1

Include the -y flag if you want to set up defaults. The default langauge is Typescript


sern plugins
1

This will display a menu selection of all installable plugins. Keep in mind that you must have a sern.config.json to use this command. If you want to view plugins, visit the repository linked above.


sern extra
1

To install extra utilities into your project
